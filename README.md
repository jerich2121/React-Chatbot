# React-Chatbot
A simple and interactive chatbot built using React with Vite for fast development and optimized builds. The project demonstrates core React concepts including components, state management, hooks, and conditional rendering. It supports real-time messaging, automatic scrolling, and a built-in dark mode toggle for better usability.
